# 🎮🔥 TALLER2-MAIN - Plataforma Gamer 

Bienvenido a **TALLER2-MAIN**, una aplicación web desarrollada con React + Vite enfocada en una experiencia moderna tipo **tecnología gamer** ⚡.
Este proyecto está estructurado para ser escalable, organizado y fácil de mantener.

---

## 🧠💻 Tecnologías usadas

* ⚛️ React
* ⚡ Vite
* 🎨 CSS moderno
* 📦 Node.js

---
